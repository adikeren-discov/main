The main repsitory of evolutionary algorithms

main.py – פונקציה ראשית הרצת סימולציות של הפרויקט
config.py – מכיל קבועים שימושיים דינמיים לפרויקט
data – תיקייה המכילה את הנתונים שהביאו לתרגיל (רצף חומצות אמינו של אינסולין, תרגום קודונים לחומצות אמינו, וציון קודונים של חיידק אי קולי)
processing.py – עיבוד הקבצים בתיקייה למבני נתונים מתאימים בקוד
candidate_solution.py – מחלקה המכילה רצפי קודונים כפתרונות לבעיה (רצפי קודונים)
mutation_engine.py – אתחול פתרונות ראשוני, ומנגנון יצירת מוטציות אקראיות לפתרונות
fitness.py – מכילה את הפונקציה לשערוך הציון של רצף לדרישות
heuristics.py – מכילה פונקציות היוריסטיות לאלגוריתמים 2, 3 (קנס על אחוז ג'י סי חריג, שיפור ציון קודונים וכד')

__init__.py – קובץ עזר הכרחי
base_strategy.py – המחלקה האבסטרקטית הראשית לאסטרטגיות למציאת פתרונות
blind_strategy.py – מחלקה אבסטרקטית (יורשת מהמחלקה הראשית הקודמת), חסרת  היוריסטיקה, מייצגת אבולוציה אקראית טהורה לאלגוריתם 1
non_blind_strategy.py – מחלקה אבסטרקטית (יורשת מהמחלקה הראשית בדומה לקודמת) משלבת היוריסטיקה לאלגוריתמים 2, 3 בלדווין ולמארקיאן.
Random_mutation_strategy.py – יורש מהמחלקה העיוורת
Baldwin_strategy.py – יורש מהמחלקה הלא עיוורת
Lamarckian_strategy.py – יורש מהמחלקה הלא עיוורת
